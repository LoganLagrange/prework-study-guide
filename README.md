# Prework Study Guide Webpage

## Description

This Prework Study Guide was created for bootcamp students going through the prework for their coding course. It contains notes about HTML, CSS, Git, and JavaScript

## Installation

N/A

## Usage

It is used by looking at the notes in each section, you can also open the Chrome Devtools console to see the Javascript running which will tell you what topics are contained within the study guide and will choose a random topic for you to study. Devtools can be accessed by pressing Command+Option+I (MacOS) or Control+Shift+I (Windows) and selecting the console tab if not already.
